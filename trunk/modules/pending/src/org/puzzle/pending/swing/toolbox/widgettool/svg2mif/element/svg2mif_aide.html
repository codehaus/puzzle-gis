SVG2MIF est un utilitaire permettant la conversion de graphismes vectoriels au format SVG vers un fond de carte au format MIF/MID (Mapinfo © Interchange).
Seules la géométrie et la position des objets sont conservées, pas les styles de présentation.

 

Les types d'objets SVG gérés et leur conversion en objets MIF :
  • polygon -> polygon (centroïde calculé comme le centre du rectangle d'encombrement).
  • path -> polyline (sauf arcs)
  • rect -> rect
  • line -> line
  • polyline -> polyline
  • circle -> ellipse
  • ellipse -> ellipse

    •  

    Par défaut, le systèmes de coordonnées du fond de carte est de type non-teerestre, avec les centimètres comme unité.
    Certains logiciels de cartographie automatique ne gérant que les objets polygones, il est possible (case à cocher) de demander la conversion des objets du fichier SVG en polygones MIF. Dans ce cas, les cercles et les ellipses sont converties en polygones rectangulaires.
    De même, certains logiciels ne gèrant pas le géoréférencement, il est possible de demander le retournement vertical du fond de carte.
